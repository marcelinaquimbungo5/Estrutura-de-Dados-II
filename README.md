# Estrutura-de-Dados-II
Repositório exclusivo para o trabalho de Estrutura de Dados, pilha e fila.

Este projecto apresenta a implementação de duas estruturas de dados fundamentais.
*Fila (Queue)
*Pilha (Stack)

As implementações foram feitas nas linguagens Java e Python, com o objectivo de demonstrar o
funcionamento básico de cada estrutura.
